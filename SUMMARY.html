🔌 Backend & integrations

StoreHub (api.storehubhq.com) → live menu, stock, prices — the same backend as your POS app, so they stay in sync.  ·  Omise/Opn → PromptPay + card.  ·  Grok (xAI) → น้องแดงค์'s open-ended brain.  ·  Telegram / Resend → staff notifications.  ·  Upstash Redis → orders, chats, wallet storage. Everything degrades gracefully — the site always works and lights up each feature as its key is added.

🚀 To publish

  1. Create a free Vercel account.
  2. Drag the project folder in (New → Project → Deploy).
  3. Add env vars: STOREHUB_STORE=dankclub, STOREHUB_TOKEN=your token, STAFF_KEY=your secret (add Omise/Grok/Telegram later).
  4. Deploy, then add domain dankbkk.com and set the DNS record shown.
  5. Open /staff.html, log in, and you're live.
